# Context Pack

## Purpose

Create a compact context pack instead of loading large files, logs, or repositories.

## Include

1. Current task.
2. Relevant files or notes.
3. Key constraints.
4. Important decisions.
5. Open conflicts.
6. Next action.
7. Do-not-read-first list.
8. Raw sources only when necessary.

## Rules

- Prefer summaries with source paths.
- Do not include secrets.
- Do not dump large logs.
- Keep context sufficient, relevant, and economical.
